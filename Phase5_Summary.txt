================================================================================
  AXIOS CONFIG (src/api/axiosConfig.js)
================================================================================

- Base URL: http://localhost:3001/api
- Request interceptor: auto-injects JWT from localStorage (key: auth_token)
- Response interceptor:
  - Passes through successful responses
  - On 401: removes token, redirects to /login (with isRedirecting flag
    to prevent multiple simultaneous redirects)

================================================================================
  ROUTING & AUTH FLOW (src/App.jsx)
================================================================================

Architecture:
  <BrowserRouter>
    <Route path="/login">       → Login page (no sidebar)
    <Route path="/*">           → AppLayout (sidebar + content)

AppLayout:
  - Sidebar with "عقاري" branding + navigation items
  - Protected Route wrapper for Dashboard:
    - No token → <Navigate to="/login" replace />
    - Token exists → renders children (Dashboard)

Login page:
  - Token exists → <Navigate to="/" replace /> (skip login if already auth'd)

Auth flow:
  1. User opens http://localhost:3000
  2. No token → ProtectedRoute redirects to /login
  3. User enters credentials → POST /api/auth/login
  4. Token stored in localStorage → navigate to /
  5. Token present → Dashboard loads → API calls succeed

================================================================================
  DASHBOARD (src/components/Dashboard.jsx)
================================================================================

Data sources (3 parallel API calls):
  GET /api/dashboard/summary    → metrics + recent payments
  GET /api/invoices?limit=15    → invoice list for table
  GET /api/payments?limit=30    → payment timestamps for Undo check

Metric Cards (grid of 4):
  1. إجمالي الدخل الشهري     - totalMonthlyIncome (accent: blue)
  2. نسبة الإشغال            - occupancyRate + occupied/total units
                               (accent: green)
  3. المستأجرون النشطون      - activeTenants count (accent: purple)
  4. المتأخر على المستأجرين  - overdueAmount + unpaid invoices count
                               (accent: red)

Recent Financial Movements Table:
  Columns: الفاتورة | المستأجر | الوحدة | المبلغ | الحالة | تاريخ الاستحقاق | إجراء
  Status badges with colors:
    - مدفوعة (Paid)        → green (#34d399)
    - مدفوعة جزئياً (Partial) → amber (#fbbf24)
    - متأخرة (Overdue)     → red (#f87171)
    - غير مدفوعة (Unpaid)  → gray (#94a3b8)
    - ملغاة (Cancelled)    → muted (#64748b)
  Undo button (↩):
    - Only visible for Paid invoices with a payment within the last 24h
    - Cross-references payment createdAt timestamps against current time
    - Shows confirmation dialog before action

State handling:
  - Loading: skeleton shimmer animation for metric cards + table rows
  - Error: centered error card with retry button
  - Empty: "لا توجد حركات مالية حتى الآن" in table body
  - Undo toast: green notification bar that auto-dismisses after 3s
