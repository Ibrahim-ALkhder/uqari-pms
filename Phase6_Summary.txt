   - Dashboard.jsx: `import api from '../../api/axiosConfig'` → correct depth
   - Login.jsx:    `import api from '../../api/axiosConfig'` → correct depth

3. Vite Proxy Configuration (client/vite.config.js):
   ```js
   server: {
     port: 5173,
     proxy: {
       '/api':     { target: 'http://localhost:3001', changeOrigin: true },
       '/uploads': { target: 'http://localhost:3001', changeOrigin: true },
     },
   }
   ```
